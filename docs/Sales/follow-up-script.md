# Follow-Up Script

Use these templates for the outreach sequence. Replace placeholders like `{{first_name}}` and `{{company_name}}` before sending.

## Day 1: Initial Outreach Template

**Subject:** Quick GitHub repo audit for {{company_name}}

Hi {{first_name}},

I took a quick look at {{company_name}}’s public GitHub presence and saw a few easy wins around security policy, CI consistency, and branch protection.

I offer a short repo audit that highlights the highest-value fixes first. If helpful, I can send a sample and the full scope.

Best,
Roman

## Day 3: Follow-Up Template (No Response)

**Subject:** Re: Quick GitHub repo audit for {{company_name}}

Hi {{first_name}},

Just following up in case my last note got buried.

I still think a small GitHub audit could help {{company_name}} tighten repo hygiene and make the public engineering story stronger. If you’d like, I can send over a short sample.

Best,
Roman

## Day 5: Final Follow-Up Template (Breakup Email)

**Subject:** Should I close the loop?

Hi {{first_name}},

I haven’t heard back, so I’ll assume timing isn’t right for now.

If GitHub governance or security becomes a priority later, I’d be happy to send a sample audit and revisit it then. Thanks for considering it.

Best,
Roman

## Day 7: Conversion Template (Interested Lead)

**Subject:** Next steps for the {{company_name}} audit

Hi {{first_name}},

Great to hear you’re interested.

To get started, I’ll just need the public repo URLs you want reviewed and any priorities you want me to focus on. Once I have that, I’ll map the findings into a clear action list and next-step recommendations.

If you want, I can also send a simple scope summary before we begin.

Best,
Roman
