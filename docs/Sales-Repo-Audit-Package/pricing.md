- Repository review and findings summary

- Prioritized recommendations
- Client-ready report for Roman to send

Best for teams that want the audit plus implementation support on the highest-priority issues.

- Everything in Basic Audit

## 3) AI Ops Retainer — $1K-$5K/month
Best for clients who want ongoing repo optimization, automation, and support.

**Includes:**
- Monthly repository health review
- Continuous improvement recommendations
- GitHub workflow and automation optimization
- AI-assisted documentation, triage, and repo ops support
- Priority handling for new repo issues and improvement requests

The Basic Audit is a low-friction entry offer. The Audit + Fix Sprint turns findings into visible outcomes, and the AI Ops Retainer keeps the repo improving after the first win.
