# Audit Report Quality Gate

Use this checklist before delivering any client-facing audit report. Do not ship the report until every item is checked.

## Pre-delivery checklist

- [ ] All critical findings have evidence (file path, line number)
- [ ] All warnings have evidence
- [ ] Security score is calculated correctly
- [ ] Executive summary is clear and non-technical
- [ ] Recommendations are actionable
- [ ] Next steps section includes pricing
- [ ] No false positives (verified by manual review)
- [ ] Report follows template (docs/Templates/audit-report-template.md)

## Delivery rule

If any item above is unchecked, the report must be revised before delivery.
