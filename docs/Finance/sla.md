# JAGM IT Company

## Service Level Agreement (SLA)

**Service:** GitHub Repository Audit

### 1. Response Time
- Audit request confirmation will be provided within **24 hours** of receiving the request.

### 2. Delivery Time
- **Basic:** delivery within **48 hours**
- **Standard:** delivery within **5 days**
- **Premium:** delivery within **7 days**

### 3. Quality Guarantee
If an audit misses a **Critical** finding that was present at the time of the audit, the client is eligible for a **full refund**.

### 4. Support
- **30 days post-delivery** support is included for clarification questions related to the audit report.

### 5. Confidentiality
- **NDA available on request** to support confidentiality requirements.

### 6. Scope and Limitations
- The SLA applies only to the agreed audit scope documented at the time of engagement.
- Any expanded scope, remediation, or follow-up implementation may require a separate agreement.
