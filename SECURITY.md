# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to security@jagm-it.company.

- Do NOT open a public GitHub issue
- Email security@jagm-it.company with details
- You will receive a response within 48 hours
- Please do not disclose the vulnerability publicly until it has been addressed

## Supported Versions

Only the latest release of each project is supported with security updates.

## Security Measures

- Dependabot enabled for dependency monitoring
- Branch protection on main branches
- Code review required for all changes
- No direct commits to main
